安装说明：

1.         本系统需要Java运行环境及数据库系统，请先确定已经安装了最新版的JRE6和数据库系统。（Java下载网址：http://www.java.com/zh_CN/download/index.jsp。数据库推荐使用MySQL 5.1，下载地址：http://dev.mysql.com/downloads/mysql/5.1.html）

2.         安装好数据库后，使用数据库自带的备份工具导入database.sql文件（该文件包含了HotelManager酒店管理系统的数据库结构和部分测试数据）。

3.         用文本编辑工具编辑database.properties文件中的如下条目：
jdbc.drivers : Server端所安装的数据库系统对应的JDBC驱动的类名
url : Server端的数据库访问地址
user：用于数据库连接的用户名
password：对应的密码
再编辑server.properties中的ServerIP，使之成为Server端得IP地址

4.         双击HotelManager.jar启动HotelManager酒店管理系统（系统管理员用户名：0；密码：0000）。

其他说明：

1.         HotelManager.chm为该系统的帮助文档，用户在使用过程中遇到问题可以参考

2.         src.zip为该项目的源代码（用Eclipse 3.4开发的）。
